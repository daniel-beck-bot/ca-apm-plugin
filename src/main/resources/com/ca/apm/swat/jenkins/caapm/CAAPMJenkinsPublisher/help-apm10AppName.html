Use this parameter to specify vertexes in APM Team center where build status needs to be published.
For e.g. "applicationName:TixChange" would publish build status info from Jenkins to all the ATC nodes that have applicaitonName as TixChange
Both attribute name and value are case sensitive
The name here has to match attribute name in APM Team Center's attribute detail pane (right side pane when you select a node) for a node or nodes
The value is the value of that attribute.
